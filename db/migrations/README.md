# db/migrations

SQL migration files for the bp-index Postgres schema, applied in date order.

Each file is idempotent (`IF NOT EXISTS`) and additive — existing tables are not altered or dropped. Run on the `production` schema of the bp-index Railway database.

## Applying a migration

```sh
# from the repo root, with .env.db loaded
PGPASSWORD=$DB_PASSWORD psql -h $DB_HOST -p $DB_PORT -U $DB_USER -d $DB_NAME \
    -f db/migrations/<file>.sql
```
